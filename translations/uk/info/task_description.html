Використовуйте Luhn algorithm для перевірки номерів кредитних карток. Кожен номер кредитної картки подається у вигляді рядка і має правильну форму з чотирьох чотиризначних блоків, розділених одним пробілом.

Для цього алгоритм включає такі кроки:

  1. Подвоїти кожну другу цифру, починаючи з крайньої правої.

  2. Якщо результат подвоєння більший за 9, додайте цифри числа.

  3. Складіть всі цифри.

  4. Перевірте, чи сума однозначно ділиться на 10.
