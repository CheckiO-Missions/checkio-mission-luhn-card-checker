Utilisez Luhn algorithm pour valider les numéros de cartes de crédit. Chaque numéro de carte de crédit est donné sous forme de chaîne et a une forme correcte de quatre blocs de quatre chiffres, séparés par un simple espace.

Dans ce but particulier, l'algorithme comprend les étapes suivantes:

  1. Doublez un chiffre sur deux en commençant par le plus à droite.

  2. Si le résultat du doublement est supérieur à 9, additionnez les chiffres du nombre.

  3. Additionnez tous les chiffres.

  4. Vérifiez si la somme est clairement divisible par 10.
