W tym konkretnym celu algorytm obejmuje takie kroki:

  1. Proszę podwoić co drugą cyfrę, zaczynając od prawej.

  2. Jeśli wynik podwojenia jest większy niż 9, proszę dodać cyfry liczby.

  3. Proszę zsumować wszystkie cyfry.

  4. Proszę sprawdzić, czy suma jest podzielna przez 10.
